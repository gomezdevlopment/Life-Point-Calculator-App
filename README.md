# Life-Point-Calculator-App
Android App for tracking life point totals in trading card games.

The app is designed for the Yu-Gi-Oh! Trading Card Game. 

Life points start at 8000 for each player. 

### Features: 
-Dice Roll\
-Coin Toss\
-Dark Mode\
-Sound FX
